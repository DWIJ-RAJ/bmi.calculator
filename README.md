# BMI Calculator App

A simple and professional BMI (Body Mass Index) Calculator application built using Python and Tkinter.

## Features
- **Dual Mode:** Supports both Metric (kg/m) and Imperial (ft/in) calculation.
- **Data Persistence:** Automatically saves your calculation history to a text file (`bmi_records.txt`).
- **Error Handling:** Robust validation to prevent app crashes from invalid inputs.
- **User-Friendly UI:** Clean interface with color-coded buttons and clear status indicators.

## How to Run
1. Make sure you have Python installed on your system.
2. Clone this repository or download the source code.
3. Run the script using the following command:
   ```bash
   python main.py
